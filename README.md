# **Alzheimer’s Disease Analysis – Tableau Dashboard**
## **Project Overview**

This project presents an exploratory data analysis of an Alzheimer’s disease dataset using Tableau Desktop Public Edition. The objective is to identify demographic patterns, health-related trends, and statistical insights through interactive visual analytics.

The dashboard enables structured exploration of disease distribution and key influencing variables.

### **Objectives**

- Perform exploratory data analysis (EDA)
- Analyze demographic distribution of Alzheimer’s cases
- Compare statistical measures (e.g., average vs. median)
- Identify patterns across health and age-related variables
- Build an interactive, insight-driven dashboard

### **Tools & Technologies**
- Tableau Desktop Public Edition
- Excel / CSV dataset
- Data preprocessing and cleaning prior to visualization

### **Dashboard Features**

* Interactive filters for segmentation

- Comparative visualizations across demographic groups

* Central tendency analysis (Average vs. Median)

- Structured layout for clear interpretability

- Multi-sheet dashboard integration
